# Contributing to InvestNova

Thank you for your interest in contributing to InvestNova! We welcome contributions from the community to help improve this AI trading platform.

## Code of Conduct

Please be respectful and inclusive in all interactions. We prioritize a positive and helpful environment.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/YOUR_USERNAME/invest-nova.git
    cd invest-nova
    ```
3.  **Create a new branch** for your feature or fix:
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Make your changes**, ensuring clean and readable code.
5.  **Commit your changes** with a descriptive message:
    ```bash
    git commit -m "Add amazing feature"
    ```
6.  **Push to your fork**:
    ```bash
    git push origin feature/amazing-feature
    ```
7.  **Open a Pull Request** (PR) on the original repository.

## Development Workflow

-   **Install Dependencies**: `npm install`
-   **Run Dev Server**: `npm run dev`
-   **Lint Code**: `npm run lint`
-   **Build for Production**: `npm run build`

## Project Structure

-   `/app`: Next.js 15 App Router pages and layouts
-   `/components`: Reusable UI components
-   `/lib`: Core logic (API clients, risk analysis, DB)
-   `/hooks`: Custom React hooks
-   `/public`: Static assets
